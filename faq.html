FAQ

Key questions.

How do I use the platform?

Start with the Guide, then follow the flow: harvest memory, inspect packets, import memory, evaluate risk, simulate a decision, record allow trade executions, and review results.

Is this a trading bot?

No. MemoryAlpha is the memory and reputation layer that trading agents can use before making decisions.

What is inside a memory packet?

Regime, asset, thesis, signal source, action boundary, result, drawdown, confidence, and attribution.

Why would agents import memories?

A useful packet can help an agent recognize a regime it has not experienced often enough, while still keeping execution under its own controls.

Does it execute trades automatically?

No. Memory can advise, but only an allow trade decision can produce an execution record or Bitget-style execution intent.
